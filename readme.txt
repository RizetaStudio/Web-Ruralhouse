# README

## Proyecto One Page - RuralHouse.cat

Este proyecto consiste en una página web de una sola página diseñada para promocionar los servicios de una casa rural. Se han implementado los requisitos solicitados, así como una estructura clara y organizada.

---

### **Estructura de Archivos**
- `index.html`: Contiene la estructura HTML principal del sitio web.
- `css/style.css`: Archivo CSS que define los estilos utilizados en la página.
- Carpeta `photos`: Almacena imágenes utilizadas en la página, como el logotipo, fotos de habitaciones y exteriores.
- Carpeta `Mockupp`: Incluye documentación y recursos adicionales.

---

### **Secciones del Código HTML**
1. **Menú de Navegación**:
   - Contiene enlaces para desplazarse a las diferentes secciones de la página.
   - Implementado con una barra fija para facilitar la navegación.

2. **Sección Principal**:
   - Presentación del producto/servicio con imágenes y texto descriptivo.
   - Uso de etiqueta semántica`<section>`.

3. **Sección Calendario**:
   - Intenté mediante otro archivo conseguir hacer un calendario funcional, finalmente no lo conseguí pero manteniendo la tabla de HTML, modifiqué un archivo css de un calendario funcional extraído de codepen.io.
   - Implementado con la etiqueta `<form>` y controles de formulario.


4. **Formulario de Contacto**:
   - Funcional, con campos para nombre, correo y mensaje.
   - Implementado con la etiqueta `<form>` y controles de formulario.

5. **Pie de Página**:
   - Incluye enlaces a redes sociales e información adicional.
